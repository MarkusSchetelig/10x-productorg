Debugging Robot in Pillars Section

Steps to Debug:

  1. Open the app at http://localhost:5176/
  2. Open Browser Developer Console (F12 or right-click → Inspect → Console)
  3. Scroll to the "Key pillars of success" section (the one with 5 image tiles)
  4. Wait 5 seconds while viewing this section
  5. You should see in console: "Enabling robot for section: pillars"
  6. Then after 1 more second: "Showing pillars robot initially"
  7. The robot should appear fixed to the bottom-right of your viewport

If Robot Doesn't Appear:

  • Check console for any errors
  • Verify the section is more than 50% visible in viewport
  • Try refreshing the page and waiting on the pillars section
  • Check if the robot appears in other sections (to verify basic functionality)

Expected Console Logs:

Enabling robot for section: pillars
Showing pillars robot initially

Robot Behavior After Appearing:

  • Scroll within the pillars section → robot disappears
  • Stop scrolling → robot reappears after 3 seconds
  • Scroll out of pillars section → robot disappears
  • Return to pillars section → robot reappears after scroll stops
